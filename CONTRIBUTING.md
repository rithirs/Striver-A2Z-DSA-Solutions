# Contributing Guidelines

Thank you for considering contributing to this repository!

This repository contains solutions to **Striver's A2Z DSA Course** in multiple programming languages. Contributions are welcome and greatly appreciated.

## How to Contribute

### 1. Fork the Repository

Click the **Fork** button at the top of the GitHub repository to create your own copy.

### 2. Clone Your Fork

```bash
git clone https://github.com/your-username/Striver-A2Z-DSA-Solutions.git
```

### 3. Create a New Branch

```bash
git checkout -b feature/problem-solution
```

### 4. Make Your Changes

You can contribute by:

- Adding solutions in new programming languages
- Improving existing solutions
- Optimizing algorithms (better time/space complexity)
- Fixing bugs or mistakes
- Improving documentation and comments

### 5. Follow the Project Structure

```text
solutions/
├── cpp/
├── python/
├── java/
└── c/
```

Place your solution in the appropriate language folder under the correct topic.

### 6. Naming Convention

- **C++:** `01-right-angled-star-triangle.cpp`
- **Python:** `01_right_angled_star_triangle.py`
- **C:** `01_right_angled_star_triangle.c`
- **Java:** `01_RightAngledStarTriangle.java`

### 7. Commit Your Changes

```bash
git add .
git commit -m "Add solution for Pattern 10"
```

### 8. Push and Create a Pull Request

```bash
git push origin feature/problem-solution
```

Then open a Pull Request on the original repository.

## Coding Guidelines

- Write clean, readable, and well-structured code.
- Add meaningful comments where necessary.
- Mention **Time Complexity** and **Space Complexity** at the top of each solution (in comments).
- Follow the existing code style and folder structure.
- Make sure your solution is correct and tested.

## Reporting Issues

If you find any bug, mistake, incorrect solution, or see an opportunity for improvement, please create an Issue with a clear description.

## Thank You

Your contributions help make this repository more useful for students and developers preparing for coding interviews and competitive programming. Every contribution counts!

**Happy Coding! 🚀**
